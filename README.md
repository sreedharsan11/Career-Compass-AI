Career Compass is a full-stack web application designed to help students and freshers
analyze their resumes against job descriptions, identify skill gaps, and plan their
career growth using structured roadmaps, project ideas, and interview preparation.

---

Key Features

- Resume vs Job Description match analysis
- Intelligent match score (out of 10)
- Skill gap analysis with explanations
- Actionable improvement tips
- Role-based career roadmaps
- Suggested project ideas (role-based)
- Interview preparation (HR + Technical)
- Simple profile page for personalization

---

Tech Stack

 Backend
- Python
- Flask
- Resume & JD parsing (PDF, DOCX, TXT)
- Modular utility-based logic

 Frontend
- HTML
- CSS
- JavaScript
- Responsive dark-themed UI
