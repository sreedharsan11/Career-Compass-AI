Role-based Roadmaps

High-level maps for multiple roles. Use these to plan your learning path and decide which skills and projects to do next.

Backend Developer

  • Language: Python / Java / Node.js
  • REST APIs, HTTP, JSON
  • Databases: SQL + basic schema design
  • Authentication & authorization
  • Project: Task manager or job portal API with DB

Frontend Developer

  • HTML, CSS, JavaScript (ES6+)
  • React basics: components, hooks
  • Responsive design & accessibility
  • API integration and JSON handling
  • Project: Admin dashboard + charts

Full Stack Developer

  • Frontend + Backend + Database
  • React + Node/Flask/Django
  • Authentication, sessions, cookies
  • Deploy full-stack app
  • Project: CRUD web app with login and search

Data Scientist

  • Python, Pandas, NumPy
  • Statistics & probability
  • ML models (sklearn)
  • End-to-end pipelines
  • Project: prediction/forecasting model

Data Analyst

  • Excel / Google Sheets
  • SQL for queries & joins
  • Power BI or Tableau dashboards
  • Basic Python for data cleaning
  • Project: sales/marketing dashboard

ML Engineer

  • Python + deep learning basics
  • TensorFlow or PyTorch
  • Model serving via APIs
  • Experiment tracking
  • Project: deploy an ML model as an API

DevOps Engineer

  • Linux & shell scripting
  • Git, CI/CD pipelines
  • Docker & basic containers
  • Cloud fundamentals
  • Project: CI/CD for a sample web app

Cloud Engineer

  • AWS / Azure / GCP basics
  • Compute, storage, DB services
  • Networking & IAM basics
  • Infra-as-code basics
  • Project: deploy full-stack app to cloud

Cybersecurity Analyst

  • Networking & OS basics
  • OWASP Top 10
  • Vulnerability scanning tools
  • Log analysis & incident response
  • Project: small security lab / CTF practice

QA / Automation Engineer

  • SDLC, STLC, test design
  • Manual testing & bug reports
  • Selenium / Playwright basics
  • Test frameworks & scripts
  • Project: automation suite for demo site

Mobile App Developer

  • Flutter / React Native basics
  • UI widgets / components
  • State management & navigation
  • Backend API integration
  • Project: simple to-do / notes mobile app

Business Analyst

  • Requirements gathering & documentation
  • Process modeling (flowcharts / UML)
  • SQL & basic data analysis
  • Communication with stakeholders
  • Project: requirement doc for a mini product
